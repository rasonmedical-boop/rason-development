엑셀 매출장 업로드
✅ 모든 연도 데이터 → 과거 매출장(legacyLedger)에 보관 — 이 탭에서 조회
🔄 같은 연도 재업로드 → 해당 연도 기존 데이터를 덮어씁니다 (중복 방지)
⚠️ 컬럼 형식: 일자 / 업체명 / 품목 / 형명 / 수량 / 단가 / 공급가액 / 세액 / 합계 / 총계 / 비고
📂

과거 매출장 조회
과거 매출장 → 현재 매출장 통합
과거 매출장(22~25년)의 데이터를 매출장에 합쳐 월별 현황·대시보드에서 전체 조회
통합 안내
🔀 통합: 과거 매출장의 데이터를 salesLedger(현재 매출장)로 복사합니다.
📋 통합 후 매출장·월별 현황·대시보드에서 전 연도 데이터를 함께 조회할 수 있습니다.
✅ 엑셀 원본 데이터를 그대로 전량 반영합니다. (중복 체크 없음)
⚠️ 같은 연도를 두 번 통합하면 데이터가 중복 추가되니 주의하세요.
🗑 과거 매출장 원본은 그대로 유지됩니다 (삭제하려면 과거 매출장 탭에서 별도 삭제).
